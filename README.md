# Tour Management - Server

This is the server-side repository for the **Tour Management System**  a complete backend solution for managing tours, bookings, and users.

The application enables users to seamlessly browse available tours, make bookings, and process secure payments.  
It also includes dedicated panels for **guides** and **administrators** to manage their respective responsibilities efficiently.


---

## ✨ Features

- **User Management:**  Registration, login, profile updates, and password management.

- **Authentication & Authorization:**  Secure API endpoints using JWT (JSON Web Token).

- **Tour Management:**  Create, update, delete, and search for tours using various filters (e.g., by division, tour type).

- **Booking System:**  Allows users to book their desired tours.

- **Payment Integration:**  Facility to pay for bookings through a secure payment gateway.

- **Guide Management:**  A system to manage information and status of verified guides.

- **Admin Dashboard:**  An admin panel for monitoring and managing the entire system.
